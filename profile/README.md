# 📱PolinTech
한국폴리텍대학 인천캠퍼스 AI융합소프트웨어과 졸업작품입니다.

## 👨‍🏫프로젝트 소개
#### ‘우리 캠퍼스만의 새로운 커뮤니티를 만들자’ 라는 목표 하나로 기획된 Polintech 앱은 기존 대학생 커뮤니티앱 내 폴리텍 카테고리 하위에 묶여있던 타캠퍼스생들과 공유된 커뮤니티가 아닌 '인천캠퍼스 재학생'이 사용할 수 있는 앱을 개발하고자 하였다.
